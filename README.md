# Delaney-Solubility-with-Descriptors-ML

Predict aqueous solubility (**logS**) from a small set of molecular descriptors (Delaney / ESOL-style features):
`MolLogP`, `MolWt`, `NumRotatableBonds`, `AromaticProportion`.

This repo includes:
- A training pipeline that builds `artifacts/model.pkl` + `artifacts/preprocessor.pkl`
- A prediction pipeline used by a small Flask web app (`app.py`)
- The dataset in `notebook/data/delaney_solubility_with_descriptors.csv`

## Quickstart (Windows)

### 1) Install dependencies
If you use the provided environment in `venv/`:

```powershell
.\venv\python.exe -m pip install -r requirements.txt
```

If you are using a different Python, make sure it has `pip` and `setuptools`.

### 2) Train (creates `artifacts/`)

```powershell
.\venv\python.exe src\pipeline\train_pipeline.py
```

Outputs:
- `artifacts/preprocessor.pkl`
- `artifacts/model.pkl`

### 3) Run the web app

```powershell
.\venv\python.exe app.py
```

Open `http://127.0.0.1:5000/` and enter the four descriptor values.

## Python API usage

```python
import pandas as pd
from src.pipeline import PredictPipeline

df = pd.DataFrame(
    {
        "MolLogP": [1.5],
        "MolWt": [300.0],
        "NumRotatableBonds": [4],
        "AromaticProportion": [0.3],
    }
)

pred = PredictPipeline().predict(df)[0]
print(pred)
```

## Project structure

- `app.py` — Flask app (uses templates from `template/`)
- `template/index.html` — UI for entering descriptors and showing the result
- `src/pipeline/train_pipeline.py` — training entrypoint
- `src/pipeline/predict_pipeline.py` — inference code used by the app/API
- `src/components/` — ingestion/transformation/training components
- `notebook/data/delaney_solubility_with_descriptors.csv` — dataset
- `notebook/catboost_training.json` — example CatBoost training config (optional)
- `artifacts/` — generated training outputs (model, preprocessor, splits)
- `catboost_info/` — generated by CatBoost when you train with CatBoost (safe to ignore/clean)

## Notes / gotchas

- If you see `No such file or directory: artifacts\\model.pkl`, run the training step to generate artifacts.
- `setup.py` requires `setuptools`. If your system Python doesn’t have it, run packaging commands with `.\venv\python.exe`.
